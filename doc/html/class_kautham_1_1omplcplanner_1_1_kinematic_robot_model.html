Detailed Description

This class represents the motion of the robot The derived classes should reimplement the operator() that defines qdot = f(q, u)

Implement the function describing the robot motion: qdot = f(q, u). This function is to be reimplemeted by derived classes defining different types of behavior The derived classes should first call this function, the parent operator() function, because it sets the dimension of the dstate.

Reimplemented in Kautham::omplcplanner::KinematicCarModel, and Kautham::omplcplanner::KinematicF16Model.
