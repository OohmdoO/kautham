Detailed Description

Qualitative representation of the geometric relation

between two elements.

This qualitative representation can be very useful for performing boolean tests such as the following:

using namespace mt;
// RelType instances
RelType type1(PARALLEL);
RelType type2(DISTANCE);
type2 = type1 | type2;
type2 |= ANTIPARALLEL;
// Boolean tests
bool test;
test = type1.test(PARALLEL); // test = true
test = type1.test(SAME_DIRECTION); // test = true
test = (type1 == PERPENDICULAR); // test = false

This datatype is based on the typesafe and generic bit flags class implemented as part of the Tiny Template Library . For more information on the subject, please refer to the following link: http://www.codeproject.com/cpp/TTLFlags.asp
