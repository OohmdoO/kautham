Detailed Description

Base class for relations between geometric elements.

The RelBase class provides the members and methods for expressing the relation between geometric elements in qualitative terms. See the of the rel_type.h header for more details.

Given the distance and/or angle between the elements it computes a #RelType value that represents the relation. Example values are "PARALLEL, COINCIDENT", "PERPENDICULAR", and "SAME_DIRECTION, DISTANCE"
