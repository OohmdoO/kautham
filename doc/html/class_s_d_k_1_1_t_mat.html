Detailed Description

This class is the abstraction of the binary matrix . This class implements the square (dxd) matrix that is used to find the sequence of samples of a d-dimensional space that satisfy that the mutual distance is maximized, i.e. the minimum distance to all the previous samples of the sequence is maximized.
