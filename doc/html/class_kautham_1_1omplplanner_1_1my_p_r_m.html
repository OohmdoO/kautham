Detailed Description

This is a class derived form the class ompl::PRM. Its purpose is to slightly change its behavior, by 1) Making the ratio fo the steps grow and expand variable. It is done in the reimplementation of the solve function 2) The function expandRoadmap is changed by myexpandRoadmap. For now they are equal, but possible changes include the distance threshold in the edges of the bounce motions and changing the number of bounce steps
