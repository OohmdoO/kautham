ob::OptimizationObjectivePtr Kautham::omplplanner::omplRRTStarPlanner::createOptimizationObjectivePMD ( )

createOptimizationObjectivePMD creates a PMD alignment Optimization Objective. If there is only a single robot, it is basicly corresponds to the PMDs of the hand synergies. If there are more than one robot it corresponds to the coupling in the x,y,z coordinates of the base.

the pointer to the OptimizationObjective
