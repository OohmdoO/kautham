Detailed Description

Computes the geometric relation between two circles.

The computed distance between elements is signed, with negative values indicating that one circle is contained in the other.

Very important note: For simplicity reasons, this clas only handles cases in which the circles are coplanar. If this condition is not met, an mt::Exception is thrown.
