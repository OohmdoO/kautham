Detailed Description

template<typename KinModel>
class Kautham::omplcplanner::omplcPlannerStatePropagator< KinModel >

This class implements the state propagator used for all the omplcPlanners. It used the EulerIntegrator class to define an integrator for the propagate function. It derives from the oc::StatePropagator and includes some functions to set and get the integration time step.
