THE KAUTHAM PROJECT

A robot simulation toolkit for motion planning and teleoperation guiding

Features

  • Multiplatform and open-source
  1. Programming language: C++
  2. 3D rendering: coin3D
  3. GUI: Qt
  4. Collision detection: PQP
  5. Graph management: Boost Graph
  6. Neighborhood: MPNN
  7. Input data: XML files
  • Modular and extensible
  1. Geometric library: freeflying robots, kinematic chains and trees
  2. Sampling library: random and deterministic sampling strategies (Halton, sdk)
  3. Planning library: gridbased planners (NF1,HF), roadmaps and all the suite of planners provided by the OMPL library (ompl.kavrakilab.org)
  4. GUI library: graphic rendering and flexible management to include new planners
