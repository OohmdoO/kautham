void Kautham::InvKinWidget::getSolution ( )
privateslot

This method calls the solve method of the associated Inverse Kinematic object. Be sure you have a target set before call it. In order to have a generic method, you may set the target using the tableChanged and the setParametersFromString method of Inverser Kinematic object.
