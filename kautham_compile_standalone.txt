Compiling Kautham
=================


Ubuntu Xenial 16.04 LTS
-----------------------

First of all we consider that you have a Ubuntu box with the version Xenial
(16.04 lts, long term suport) [1]. 

$ sudo add-apt-repository ppa:deb-rob/ros-xenial

$ sudo apt-get update

$ sudo apt-get install build-essential cmake libsoqt4-dev libcoin80-dev libassimp-dev libboost-system-dev libboost-serialization-dev libboost-thread-dev libode-dev libfcl-dev libarmadillo-dev libpugixml-dev freeglut3-dev libompl-dev


Ubuntu Trusty 14.04 LTS                     
-----------------------

First of all we consider that you have a Ubuntu box with the version Trusty
(14.04 lts, long term suport) [2].

$ sudo add-apt-repository ppa:deb-rob/ros-trusty

$ sudo apt-get update

$ sudo apt-get install build-essential cmake libsoqt4-dev libcoin80-dev libassimp-dev libboost-system-dev libboost-serialization-dev libboost-thread-dev libode-dev libfcl
